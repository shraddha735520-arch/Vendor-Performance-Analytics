# Vendor Performance Analytics
**Project Overview**

Built an end-to-end Vendor Performance Analytics solution using SQL and Python to analyze vendor purchasing behavior, sales performance, profitability, inventory efficiency, and procurement trends.

The project combines SQL-based data aggregation with Python-based exploratory data analysis (EDA) to generate actionable business insights for procurement and inventory optimization.

**Tech Stack**
- Python
- SQL (SQLite)
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
**Business Problems Solved**
- Identify top-performing vendors
- Analyze purchase contribution
- Evaluate gross profit and profit margins
- Study inventory turnover
- Detect procurement inefficiencies
- Analyze bulk purchasing discounts
- Compare top vs low-performing vendors
- Perform hypothesis testing
- Confidence interval analysis
**Key Insights**
- Top vendors contribute the majority of purchases.
- Bulk purchasing significantly reduces unit cost.
- Purchase Price and Actual Price are highly correlated.
- Gross Profit strongly depends on Sales Dollars.
- Low-performing vendors maintain higher margins but lower sales.
- Procurement opportunities exist by consolidating vendor purchases.
